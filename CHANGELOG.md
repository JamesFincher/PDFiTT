# Changelog

## 1.0.0 - 2026-04-29

Initial open-source release.

- Added self-hostable Markdown-to-PDF browser app.
- Added direct PDF API at `/api/pdf`.
- Added Streamable HTTP MCP endpoint at `/mcp`.
- Added MCP tools `render_markdown_pdf` and `get_setup_instructions`.
- Added portable agent skill in `skills/pdfitt-markdown-pdf`.
- Added LLM-optimized setup docs in `llms.txt` and `AGENTS.md`.
- Added hosted `/llms.txt` entry point for LLM crawlers and agent workflows.
- Added self-hosting guide in `SELF_HOSTING.md`.
- Added full system documentation in Markdown and generated PDF formats.
